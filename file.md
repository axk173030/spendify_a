❯ what is flask in python                                                       

⏺ Flask is a lightweight web framework for Python. It lets you build web
  applications and APIs with minimal boilerplate.

  Key things it provides:
  - Routing — map URLs to Python functions (@app.route('/home'))
  - Request/response handling — read form data, JSON, query params, set HTTP
  status codes
  - Templating — render HTML with Jinja2 templates
  - Dev server — built-in server for local development

  It's called "micro" because it gives you the essentials and lets you pick your
  own tools for things like databases and auth, unlike Django which is more
  batteries-included.
